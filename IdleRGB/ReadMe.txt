This application has only been tested with Corsair K70 and Corsair Scimitar.
Changing LEDs is disabled by default on both mouse and keyboard.
"enableKeyboard"/"enableMouse" must be set to "True" in "IdleRGB.exe.config", depending on what periferal you have.
If your keyboard doesn't have media LEDs, "enableMedia" must be set to "False" in "IdleRGB.exe.config".
Colors can be changed in "IdleRGB.exe.config".
Time before idle can be in settings or in "IdleRGB.exe.config".

Corsair CUE needs to be running with SDK enabled.
