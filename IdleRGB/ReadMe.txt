IdleRGB

Changes LEDs on Corsair keyboard, mouse, headset, and mousemat.
Turns off all LEDs (or to a specified color) after a specified time of no activity from keyboard and mouse.
All the LEDs will also change to a another specified color when Caps Lock is activated.
No changes are made to media, win lock, or brightness LEDs for the devices that has them.
Idle time and colors can be changed by right clicking tray icon.

This application has ONLY been tested with the Corsair K70 RGB and the Corsair Scimitar.
It should however work with any of the Corsair devices supported by the CUE SDK.
Corsair CUE needs to be running with the SDK option enabled.
